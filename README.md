# **LittleLang**
> ## **Simple programming language, strongly inspired by JavaScript**
>
> *Written in Java*
# Features
 - No loops
 - No conditionals
 - Very untested

# Programs
## __`print.js`__
```js
print("Hello world!")
```
### AST:
```
CallExpression:
  Callee:
    Identifier: 'print'
  Arguments:
    StringLiteral: 'Hello world!'
```
### Output:
```
Hello world!
```
